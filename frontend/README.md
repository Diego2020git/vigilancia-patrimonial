# Frontend - Vigilância Patrimonial

## Stack

- React
- Vite
- TypeScript

## Executar

```bat
npm install
npm run dev
```

A aplicação consome API em `http://localhost:8000`.

## Telas

- Login
- Dashboard
- Unidades / Moradores / Funcionários / Financeiro / Agenda / Coberturas / Tickets / Rondas / Configurações / Auditoria
- Minhas Coberturas / Minha Unidade / Pagamentos
